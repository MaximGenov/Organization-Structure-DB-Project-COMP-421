In this Project, we attempt to simulate a real-world corporate Database System
of a firm's employee structure. In addition to defining a partition
of its employees with DDL's, this application shall support 
functionalities and commands on the Database entity sets (tables)
through SQL queries.

The project environment is IBM DB2 and queries are all in
its corresponding SQL syntax. The nature of the queries
range from simplistic to complex analytics, as well as views.

A replica of our database could be generated through bash scripts
that are part of the project structure. We shall, under no uncertain terms,
thwart anyone who is interested in building on our project from
any access of our source code nor be taking legal action over copyright infringement
as long as this README file is included in their project structure.
